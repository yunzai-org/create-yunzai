[图片开发文档](https://github.com/lemonade-lab/react-puppeteer)

## 使用方法

- 打包

```sh
yarn build
```

- 发布

```sh
npm login
```

> package.json
> 删除 "private": true

```sh
npm publish
```

## 图片开发

- 浏览器

```sh
yarn img-dev
```

## 注意

> to a dynamic import() which is available in all CommonJS modules.

```sh
rm -rf yarn.lock
rm -rf node_modules
```
