## 使用方法

- 依赖

```sh
npm install yarn@1.12.1 -y
yarn
```

- 启动

```sh
npm run app
# /
npm run dev
```

- 打包

```sh
npm run build
```

- 发布

```sh
npm login
```

> package.json
> 删除 "private": true

```sh
npm publish
```
